introduction

  • 바닐라 자바스크립트, 자료구조, 알고리즘, 운영체제 등 배우고 느낀 것을 기록하기 위해 개발 블로그를 시작했습니다. 올해 5월 11일 첫 게시글을 시작으로 지금까지 80개의 글을 작성하였고, 10월에는 총 방문 수 1,000명을 넘었습니다.
  • 또한, 문제 해결능력 향상을 위해 프로그래머스라는 사이트를 이용하여 총 98개의 문제를 풀었습니다. 코딩테스트 문제풀이는 주로 쓰이는 ES6 문법이 어떤 것이 있는지 알 수 있게 해주었고, 특히 spread 연산자와 객체 디스트럭터링(Object destructuring)은 코드 가독성을 높이는데 많은 도움을 주었습니다. 이 외에도, 코딩테스트 문제풀이는 문제를 좀 더 빠르고 정확하게 풀기 위해 자료구조 이론을 다시 공부할 수 있는 기회를 만들어주었습니다.
  • 웹 개발의 전체적인 맥락과 스킬을 향상시키기 위해 프론트앤드/백엔드 스택을 가리지 않고 공부했습니다. 특히 Wetube 프로젝트는 Node.js, 웹팩, Pug, Mongo DB등을 이용하여 유튜브 클론 사이트를 만든 것으로, 저에게 풀스택 개발 경험을 주었고, stateless 기반의 개발 이해도 향상에 도움을 주었습니다.
  • 이 외에도 YUNFLIX 프로젝트는 오직 리액트를 이용해서 만든 프론트앤드 프로젝트로, 저에게 리액트 훅의 장점을 알려주었습니다. 클래스 기반의 바닐라 리액트를 이용하는 것보다 리액트 훅을 이용하면 코드의 길이도 전체적으로 짧아질 뿐더러, 특히 useReducer, useContext, useEffect는 비동기 API 데이터와 state 관리에 편리성을 주었습니다.
